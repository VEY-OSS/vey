# Security Policy

## Supported Versions

Security fixes will be backported only to the LTS versions of each application.

See [Long-Term Support](doc/long-term_support.md) for supported LTS versions.

## Reporting a Vulnerability

Please report security bugs
by [opening a draft security advisory](https://github.com/VEY-OSS/vey/security/advisories/new) in GitHub.

Please do **not** create a public GitHub issue.
