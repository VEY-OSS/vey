.. _log_resolve_c_ares:

******
c-ares
******

Error logs generated by resolvers of type ``c-ares``.

Most fields in this log correspond directly to resolver configuration.

bind_ipv4
---------

**optional**, **type**: list of ip string

The local IPv4 bind address list.

bind_ipv6
---------

**optional**, **type**: list of ip string

The local IPv6 bind address list.

server
------

**required**, **type**: list of socket address

The configured DNS server list.
