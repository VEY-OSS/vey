.. _log_resolve_hickory:

*******
hickory
*******

Error logs generated by resolvers of type ``hickory``.

Most fields in this log correspond directly to resolver configuration.

bind_addr
---------

**optional**, **type**: ip or interface name string

The bind IP address or interface name used for the socket.

server
------

**required**, **type**: list of ip address

The configured DNS server address list.

server_port
-----------

**optional**, **type**: port number

The configured DNS server port.

encryption
----------

**optional**, **type**: string

The DNS encryption method in use.
