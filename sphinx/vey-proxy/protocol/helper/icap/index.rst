####
ICAP
####

``vey-proxy`` uses ICAP helper services for ``REQMOD`` and ``RESPMOD``
adaptation.

.. toctree::
   :hidden:

   headers
   http
   h2
   imap
   smtp

- headers

  Documents the common ICAP request headers added by ``vey-icap-client``. See
  :doc:`headers`.

- http

  Describes what is needed to enable ICAP for HTTP/1.x. See :doc:`http`.

- h2

  Describes what is needed to enable ICAP for HTTP/2. See :doc:`h2`.

- imap

  Describes what is needed to enable ICAP for IMAP. See :doc:`imap`.

- smtp

  Describes what is needed to enable ICAP for SMTP. See :doc:`smtp`.
