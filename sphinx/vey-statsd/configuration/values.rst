******************
Common Value Types
******************

The reusable configuration value types referenced by ``vey-statsd`` are
documented in the shared ``vey-values`` Sphinx project.

See :external+values:doc:`configuration/values/index` for the common value
reference.

The shared value reference also includes availability notes so you can tell
which shared value types are supported by ``vey-statsd`` and which ones are
only used by other VEY applications.
